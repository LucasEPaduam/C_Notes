
# C_Notes
Programa desenvolvido em linguagem C para criação, edição e armazenamento de notas rápidas.
Uma aplicação de console onde notas são salvas em arquivos txt e podem ser lidas pelo próprio aplicativo.
Disciplina de LP1.
